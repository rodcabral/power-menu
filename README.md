## Power Menu

Simple menu to choose between shutdown, reboot or lock.

- [x] Shutdown
- [x] Reboot
- [x] Lock

![Example](example.png)

### Install

Dependencies: make, sdl2, sdl2_ttf, xsecurelock and JetBrainsMono Nerd Font

```git clone https://github.com/rodcabral/power-menu && cd power-menu && make install```
